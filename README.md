## 1) Setup & Interface
- **Ribbon essentials:** Home, Insert, Design, Layout, References, Mailings, Review, View, **Developer** (enable via `File → Options → Customize Ribbon`).
- **Quick Access Toolbar (QAT):** Add **Show/Hide ¶**, **Style Inspector**, **Insert Page Number**, **Format Painter**, **Clear Formatting**.
- **Show/Hide (¶):** Visualize spaces (•), tabs (→), paragraph marks (¶), page/section breaks.
- **Views:** Use **Print Layout** usually; **Draft** for long style/TOC work.

---

## 2) Document Layout & Page Setup
- **Margins:** `Layout → Margins → Custom`. Typical exam: 2.54 cm unless specified.
- **Orientation & Paper:** Portrait/Landscape; A4.
- **Indentation & Spacing:** Use **Paragraph** dialog (not spaces).
- **Break types:**
  - **Page Break:** new page, same formatting.
  - **Section Break:** allows different margins, columns, headers/footers, numbering.

> **Pro tip:** Do **Page Setup first** to avoid breaking tabs, lists, and tables later.

---

## 3) Text, Paragraphs & Styles (the core skill)
- **Use styles, not manual formatting.** (Heading 1/2/3, Normal, Quote.)
- **Modify style:** Right-click style → **Modify** → set font/size/spacing; turn **Automatically update** **OFF**.
- **Spacing:** Set **Before/After** in styles; toggle “Don’t add space between paragraphs of the same style” if needed.
- **Line spacing:** 1.15 or per instruction.
- **Line & Page Breaks (Paragraph → Line and Page Breaks):**
  - **Keep with next** (for headings).
  - **Keep lines together** (avoid split paragraphs).
  - **Page break before** (new chapters).
- **Clear formatting quickly:** 
  - Character: **Ctrl+Space**
  - Paragraph: **Ctrl+Q**
  - Ribbon: **Clear All Formatting**

**Style Inspector (advanced):** Home → Styles pane → **Style Inspector** to spot direct formatting conflicts.

---

## 4) Headers/Footers, Page Numbers & Sections
- **Insert header/footer:** `Insert → Header/Footer → Edit`.
- **Different First Page / Odd & Even:** Header & Footer Tools → Design.
- **Page numbers:** `Insert → Page Number`; **Format Page Numbers…** to restart/choose numeral style.
- **Sections:** `Layout → Breaks → Next Page`; toggle **Link to Previous** off to vary headers/footers across sections.

> **Typical exam pattern:** Roman numerals (i, ii, iii) for front-matter; restart at 1 for main content.

---

## 5) Lists (Bulleted, Numbered, Multilevel)
- Apply via Home → Bullets/Numbering.
- **Define New Multilevel List:** for 1., 1.1, 1.1.1; link levels to Heading styles.
- **Restart numbering:** Right-click a number → **Restart at 1**.
- **Alignment:** In Define dialog, set **Aligned at** and **Tab after**.

> **Pitfall:** Don’t simulate indents with spaces—use proper list levels.

---

## 6) Tabs & Leader Dots
`Home → Paragraph launcher → Tabs…`
- Set positions (e.g., **8.5 cm center with Leader 2**, **11 cm left**). Click **Set** after each.
- Leaders: 1 (dots), 2 (dashes), 3 (solid line), 4 (thick line).
- Clear inherited tabs when needed.
- Press **Tab** to jump to next stop; type text; leader fills automatically.

---

## 7) Tables in Word
- **Insert → Table**.
- **AutoFit:** to contents (data) or fixed width (forms).
- **Design/Layout tabs:** Shading, Borders, Repeat Header Rows, Cell Margins.
- **Sort:** `Layout → Sort` (text/number/date).
- **Formulas:** `Layout → Formula` (e.g., `=SUM(ABOVE)`); update with **F9**.
- **Split table:** **Ctrl+Shift+Enter**.
- **Repeat heading rows** for multi-page tables.

> **Avoid:** Nested tables unless asked. Use Table Properties for alignment (not spaces).

---

## 8) Graphics: Pictures, Shapes, Icons, SmartArt, Screenshots
- **Insert:** Pictures (Device/Online), Icons, Shapes, SmartArt.
- **Alt Text** for accessibility.
- **Compress Pictures:** Picture Format → Compress (e.g., 220 ppi).
- **Screenshots:** Insert → Screenshot/Screen Clipping.
- Keep styling consistent and minimal for neatness.

---

## 9) Positioning & Wrapping
- **Layout Options** on image: In Line with Text, **Square**, Tight, Through, Top & Bottom, Behind/In Front.
- **Fix position on page** vs **Move with text**.
- **Align:** Picture Format → Align (to Page/Margins/Relative).
- **Group** objects to move as one.
- **Object anchors:** enable in `File → Options → Display`.

> **Rule of thumb:** Use **In Line with Text** unless the memo specifies floating layout.

---

## 10) Columns, Hyphenation & Control
- **Columns:** `Layout → Columns` (set count/spacing/line between).
- **Column breaks:** `Layout → Breaks → Column`.
- **Hyphenation:** `Layout → Hyphenation` (Auto/Manual).
- Combine with **Keep** options to avoid awkward breaks/widows/orphans.

---

## 11) Referencing: Footnotes/Endnotes, Captions, Cross-References
- **Footnote/Endnote:** `References → Insert Footnote/Endnote`. Set number format/location via Footnote dialog.
- **Captions:** `References → Insert Caption` (Figure/Table/Equation). Use **Numbering…** and **Include chapter number** if required.
- **Cross-References:** Insert dynamic refs to **Heading**, **Figure**, **Page number**, etc. Update with **F9**.

> **Exam classic:** “Insert an endnote on ‘NASA’, format a, b, c, with text ‘America’.” Use the Footnote/Endnote dialog to change number format.

---

## 12) Automatic Tables: TOC, TOF, LOT
- **TOC:** `References → Table of Contents → Automatic`. Built from Heading styles. Update with **F9** (entire table).
- **TOF/LOT:** First add captions (Figure 1, Table 1). Then `References → Insert Table of Figures`. Choose label and leader.

> Keep caption labels consistent (e.g., “Figure 1:”).

---

## 13) Citations & Bibliography (Harvard basics)
- Word’s built-ins (APA/MLA) may differ from local Harvard requirements.
- In exams, you’ll often **type Harvard references manually**:
  - **In-text:** `(Surname, Year)`
  - **Reference list:** `Author S. (Year). Title. Publisher.`
- Apply **Hanging indent** (Paragraph → Special: Hanging 1.27 cm) and sort A→Z.

---

## 14) Find/Replace & Format Painter
- **Ctrl+F** (Navigation Pane), **Ctrl+H** (Replace).
- **Format-aware Replace:** More → **Format…** (Font/Paragraph/Style).
- **Wildcards:** Check **Use wildcards** for advanced patterns (e.g., multiple spaces → single space).
- **Format Painter:** Double-click to paint multiple targets; **Esc** to stop.

---

## 15) Mail Merge (Letters, Labels, Envelopes)
- **Mailings → Start Mail Merge**.
- **Select Recipients:** Excel/CSV with clean headers.
- **Insert Merge Fields**, **Preview Results**, **Finish & Merge**.
- **Labels/Envelopes:** Choose vendor/product code; verify page size.

> **Numeric format switch:** `\# "R #,##0.00"` inside a merge field for currency.

---

## 16) Forms (Developer Tab)
- Enable **Developer** tab.
- Use **Rich/Plain Text**, **Date Picker**, **Combo Box**, **Check Box** content controls.
- **Restrict Editing** → Filling in forms; add password if needed.
- **Legacy Tools** for older memos if specified.

---

## 17) Proofing, Accessibility & Inspect Document
- **Spelling & Grammar:** **F7**.
- **Read Aloud:** Review → Read Aloud.
- **Accessibility Checker:** Review → Check Accessibility (Alt Text, heading order, contrast).
- **Inspect Document:** File → Info → Check for Issues (remove personal data/hidden text).

---

## 18) Track Changes, Comments & Compare
- **Track Changes:** Review → Track Changes; use **Simple Markup**.
- **Accept/Reject** as required; filter **Show Markup**.
- **Comments:** New Comment (**Ctrl+Alt+M**).
- **Compare:** Review → Compare (Original vs Revised).

---

## 19) Templates, Themes & Building Blocks
- **Themes:** Design → Themes (fonts/colors/effects).
- **Templates:** Save as **.dotx** (or **.dotm** for macros).
- **Quick Parts / Building Blocks:** Save reusable cover pages, headers, text blocks.

---

## 20) Fields (Date, Filename, Page X of Y, IF, Seq)
- **Insert Field:** Insert → Quick Parts → Field…
- Toggle codes: **Alt+F9**; update: **F9**.
- **Examples:**
  - **Page X of Y:** insert `PAGE` and `NUMPAGES` fields.
  - **Conditional text:** `IF { PAGE } > 1 " (Continued)" ""`
  - **Custom numbering:** `SEQ` fields.

---

## 21) Macros (awareness), Security & Trusted Locations
- **Record Macro:** View → Macros → Record Macro; assign to QAT/keyboard.
- **Trust Center**: manage macro security/trusted locations.
- For exams, **awareness** is enough unless the task requires macros.

---

## 22) Exporting & Printing (PDF/XPS, quality, bookmarks)
- **Save As → PDF**. Options: **Document structure tags** (accessibility), **Create bookmarks** from headings.
- **Print:** correct printer, **Print Markup** off unless requested, scale 100%.

---

## 23) Keyboard Shortcuts: The Matric “Fast-Path”
- **Heading styles:** Alt+Ctrl+1/2/3
- **Clear formats:** Ctrl+Space (character), Ctrl+Q (paragraph)
- **Find/Replace:** Ctrl+F / Ctrl+H
- **Format Painter:** Ctrl+Shift+C (copy), Ctrl+Shift+V (paste)
- **Repeat last action:** F4
- **Non-breaking space/hyphen:** Ctrl+Shift+Space / Ctrl+Shift+-
- **Footnote/Endnote:** Alt+Ctrl+F / Alt+Ctrl+D
- **Update/Toggle fields:** F9 / Alt+F9
- **Go To:** Ctrl+G

## 24) Exam Strategy & Marking-Friendly Formatting
1. **Follow the memo exactly.** If it says “apply style X,” don’t fake it.
2. **Structure first:** margins, sections, styles → then content.
3. **Use Show/Hide (¶)** to reveal invisible mistakes.
4. **Styles over manual formatting** for TOC/cross-refs consistency.
5. **Place objects with Layout Options** (don’t drag randomly).
6. **Save versions** (v1, v2…) and back up.

---

## 25) Troubleshooting: 50 Common Errors & Quick Fixes
1. **TOC won’t update:** Headings aren’t real Heading styles → apply correct style, press **F9**.
2. **Page numbers restart oddly:** Check **Format Page Numbers → Continue from previous**.
3. **Need different header:** Turn **Link to Previous** off (in that section).
4. **Leader dots missing:** Define **Tabs…** with leader and position; press **Set**.
5. **Bullets misaligned:** Define New Multilevel List; set **Aligned at / Tab after**.
6. **Image jumps:** Use **In Line** or **Square** + **Fix position on page**.
7. **Broken caption numbering:** Ensure consistent label; enable **Include chapter number** if needed.
8. **Cross-ref shows “Error”:** Update fields (**F9**) or re-insert if target moved/deleted.
9. **Table row splits:** Table Properties → Row → uncheck **Allow row to break across pages**.
10. **Inconsistent spacing:** Clear formatting (Ctrl+Space/Ctrl+Q), reapply style.
11. **Ugly hyphenation:** Adjust Hyphenation or use **Keep with next**/**Keep lines together**.
12. **Endnotes wrong numbering:** Footnote dialog → set **Number format/Restart** properly.
13. **Mail Merge currency wrong:** Add numeric switch `\# "R #,##0.00"`.
14. **Header only first page:** Use **Different First Page** intentionally.
15. **TOF/LOT empty:** Ensure captions exist with correct label.
16. **Hidden section breaks:** Turn on Show/Hide (¶) to locate/manage.
17. **File too large:** Compress pictures; remove embedded objects.
18. **Spellcheck disabled:** Review → Language → Set Proofing Language (uncheck “Do not check…”).
19. **Styles auto-change:** Turn **Automatically update** off in Modify Style.
20. **Paragraph splits at page end:** **Keep lines together**.

_(…extend this list as you practice and spot patterns.)_

## 26) Mini-Drills & Practice Tasks
**A. Styles & TOC (15 min)**  
- Paste a 2-page unformatted text. Create Title, Heading 1/2/3 styles. Apply them. Insert an automatic TOC.

**B. Tabs & Leaders (10 min)**  
- Build a two-column leaders list: left item names; right page numbers using **center 8.5 cm (leader 2)** and **left 11 cm**.

**C. Tables (15 min)**  
- Create a 5×6 table. Add headings. Sort by a numeric column. Compute **SUM(ABOVE)** for the last row.

**D. Captions & TOF (10 min)**  
- Insert two images; caption as **Figure 1** and **Figure 2**; insert a **Table of Figures**.

**E. Mail Merge (20 min)**  
- Merge a letter with 10 recipients (Excel). Include a conditional VIP line using an **IF** field.

**F. Forms (Developer) (15 min)**  
- Build a one-page form (name, date, checkbox list). **Restrict Editing** → Filling in forms.

- Confirm leader tabs via the Tabs dialog (for teaching screenshots).
- For endnotes with custom numbering (a, b, c), demonstrate Footnote/Endnote dialog settings.
